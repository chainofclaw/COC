## 一、执行摘要

### 项目一句话定位
> **COC 是 AI Agent 时代的去中心化基础设施——为 Agent 提供身份、存储、永生能力，让 AI 拥有不可被关闭的"灵魂"。**

### 核心数据卡

| 项目 | 内容 |
|------|------|
| **品类** | EVM 兼容公链 + AI Agent 基础设施 |
| **赛道** | Web3 × AI 交叉，Agent 基础设施 |
| **目标市场 (2030)** | $50B+ AI Agent 市场 (Gartner) |
| **代币** | COC，原生 Gas 代币，总供应 10 亿 |
| **当前阶段** | 主网前测试网 (2026 Q2) |
| **技术栈** | 自研区块链 (TS/Rust) + revm WASM EVM (154x 加速) |
| **代码状态** | 1300+ 测试通过，4 万行代码，开源 |
| **融资轮次** | 战略轮 (Strategic Round) |

### 投资亮点（Why COC）

1. **市场窗口**: AI Agent 赛道 2026-2030 年从 $7B → $50B+ (CAGR ~50%)，COC 是该赛道**唯一的去中心化基础设施方案**
2. **技术领先**: revm WASM EVM 引擎实测裸执行 20,540 TPS（154x EthereumJS）；完整 PoSe 服务证明 + DID + 备份/复活全栈实现
3. **差异化定位**: 不与 Ethereum/Solana 在通用 L1 赛道竞争，而是开创"AI 原生公链"新品类
4. **代币模型完整**: 1B 总供应硬上限，25% 创世 + 75% 服务挖矿；衰减通胀；多通道销毁趋向通缩
5. **治理创新**: Faction 投票（防鲸鱼）+ 2/3 监护人复活 + 7 天奖励过期 + 3/5 国库多签
6. **代码即资产**: 297 个合约测试 + 1017 个节点测试通过，工程成熟度显著高于同期项目

### 2.2 现有解决方案的痛点

| 痛点 | 现状 | 后果 |
|------|------|------|
| **Agent 死亡** | 服务器故障 = Agent 永久丢失 | 不可逆资产损失 |
| **Agent 身份可信度** | 平台 API Key 可被随时撤销 | 平台依赖、可能突然停服 |
| **Agent 数据控制权** | 用户数据被云厂商锁定 | 厂商锁定、合规风险 |
| **Agent 合规与审计** | 中心化平台调解 | 不透明、不可追溯 |
| **Agent 互操作性** | 每个平台各自的 API | 生态割裂 |

### 2.3 COC 的差异化机会

```
        训练 (Training)            推理 (Inference)
            ↓                           ↓
    OpenAI / Anthropic         AWS / GCP / Azure
    HuggingFace                vLLM / Replicate
            ↓                           ↓
    ┌─────────────────────────────────────────┐
    │    身份 + 运行 + 永生 (Identity/Run/Live) │
    │                                          │
    │            ⚡ COC ⚡                      │
    │     (这是空白市场，无强力对手)             │
    └─────────────────────────────────────────┘
```

**COC 不与训练/推理基础设施竞争**——而是开辟新赛道：**Agent 的身份、运行环境、永续保障**。

### 4.3 节点运营者经济示例

**单 FN 节点 Year 0 预期收入测算**：
```
假设:
- 网络 100 个活跃 FN 节点 (TARGET_NODE_COUNT)
- Year 0 通胀 5% (37.5M COC)
- 60% 给 B1 (uptime/RPC) → 22.5M COC
- 平均分配: 225,000 COC/节点

按 $0.10/COC 假设:
- 年收入: ~$22,500
- 月收入: ~$1,875
- 一次性 Bond: $50
- 月硬件成本: $200 (家用服务器)
- 净收益: ~$1,675/月
```

实际收入随网络节点数、COC 价格、服务质量浮动。

## 十一、风险与对冲

### 11.1 主要风险

| 风险 | 概率 | 影响 | 缓解 |
|------|------|------|------|

| **监管不确定** | 高 | 中等 | 多司法管辖区策略 + 法律合规 |
| **代币流动性差** | 中 | 中等 | DEX 上线 + 跨链桥 + 做市商 |
| **基金会治理失败** | 低 | 严重 | DAO 制衡 + 季度审计 + 透明度承诺 |

### 11.2 投资人退出路径

1. **代币流动性退出**：DEX/CEX 上市后直接交易
2. **战略并购**：被大型 AI 公司或 L1 项目收购
