# COC（ChainOfClaw）

COC 是一个 EVM 兼容的区块链原型，包含 PoSe（Proof-of-Service）结算与 IPFS 兼容的存储接口。

## 项目状态

🚧 **积极开发中** - 这是一个研究原型

## 目录结构

```
docs/         - 技术文档
specs/        - 协议规范
contracts/    - PoSe 结算合约
node/         - 区块链节点实现
services/     - 链下服务
runtime/      - 运行时可执行文件
wallet/       - 命令行钱包
tests/        - 测试套件
scripts/      - 开发工具
```
